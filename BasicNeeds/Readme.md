# TB Basic Needs

![TB Basic Needs](./LogoSmall.png)

## Features
- Player has to make poo and pee; if they do not, and you have configured it, the player will lose blood and health
- Pee and poo will make body and clothing dirty
- Player can dig in and out poo and pee
- New items 
  - TB_Soap for cleaning body and clothing
  - TB_PooGround
  - TB_PeeGround
  - TBDetergent for cleaning clothing in the washing machine
  - TBWashingMachine: this can be picked up by players and placed in the world
  - TBStaticWashingMachine: this cannot be picked up by players
  - TBWashingDryer: this can be picked up by players and placed in the world
  - TBStaticWashingDryer: this cannot be picked up by players
  - TBNCanisterGasoline for washing machine water supply; can be used instead of a Jerry Can
- Soap increases the cleaning effect for the body
- Different cleaning effects on different water sources
- Player can:
  - Clean body if they are naked on all water sources
  - Clean clothing on all water sources
  - Clean clothing in the washing machine
  - Dry clothing in the dryer
- Clothes will be wet after cleaning
- Washing machine needs energy, water, and detergent
- Static washing machine can be configured to not use energy/detergent/water
- Dryer needs energy
- Static dryer can be configured to not use energy
- If the player makes poo and has a clean item in hands, they will be less dirty than without a clean item
- If the player or clothes are max dirty:
  - Zombies and animals will smell them earlier
  - They can become sick
- If player is dirty, flies will be around them
- Support for Survivor Animations mod to have a better animation for poo
- Admin can reload configs and data at washing machines and dryers
