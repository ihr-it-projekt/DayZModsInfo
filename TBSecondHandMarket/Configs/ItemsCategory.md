# ItemsCategory.json
This category config helps you to filter the Items on Bulletin Borad and Globale Market Stall

````json lines
{
    "version": "1",
    "isInitialized": 1,
    "categories": [
        {
            "name": "Name of your Category 1"
            "items": [
                "Item_Type_1",
                "Item_Type_2"
            ]
        },
        {
            "name": "Name of your Category 2"
            "items": [
                "Item_Type_3",
                "Item_Type_4"
            ]
        }
    ]
}
````
